# Snake Game Web

Jogo da cobrinha em HTML, CSS e JavaScript.

## Como executar

1. Abra o terminal na pasta `codespaces-blank`.
2. Instale dependências:

```bash
npm install
```

3. Inicie o servidor:

```bash
npm start
```

4. Abra no navegador:

```text
http://127.0.0.1:8000
```

## Compartilhar com outra pessoa

Basta enviar toda a pasta do projeto (`index.html`, `styles.css`, `script.js`, `package.json`, `README.md`).

A pessoa só precisa executar:

```bash
npm install
npm start
```

## Recursos do jogo

- Controle por teclado e botões na tela
- Pontuação e recorde
- Upgrade de escudo, turbo e pontos em dobro
- Inimigos móveis e eventos aleatórios
- Layout responsivo para rodar no browser
